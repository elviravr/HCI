Arduino.

Arduino is an open-source electronics platform for building prototypes. Arduino consists of an open source micro-controller and a software environment to program it in. The Arduino assignments we did, run with software (e.g. Processing) on our computer. The Arduino micro-controller can receive input from sensors and send output to actuators.
